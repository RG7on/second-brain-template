# knowledge/ — route map

Read this first. It routes any question to the right folder in one hop.

| I'm looking for… | Go to |
|---|---|
| A choice that was made, and why | `decisions/` (dated, one per file) |
| How to do something, durable facts | `reference/` |
| Everything about a subject | `topics/` hub page, then follow links |
| A project's state and history | `projects/<name>/overview.md` |
| A person | `people/` |
| Personal/life areas | `life/` |
| What happened on a date | `journal/YYYY/YYYY-MM-DD.md` |
| Unprocessed quick captures | `inbox/` |
| Superseded (old) versions | `archive/` — history only, never current |
| Encrypted sensitive notes | `vault/` — `.age` only; see `setup/runbooks/vault.md` |
| Small binary files | `attachments/` — 1MB cap per file |

Search protocol and note-writing rules live in `CLAUDE.md` at the repo root.
Topic vocabulary lives in `topics.yaml` next to this file.
